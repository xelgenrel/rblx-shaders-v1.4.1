## 🎯 How rblx-shaders-v1.4.1 Works

Roshade modifies how your game displays graphics. It uses shader files to change color tones and sharpness, plus adds effects that mimic natural light and shadows. This process happens outside the game itself, making it easy to toggle effects without altering game files.

You can switch between different presets to find the look you prefer. The included presets cover styles like:

- **Sharp:** Boosts edge clarity and detail  
- **Vivid:** Enhances color and saturation  
- **Atmospheric:** Adds fog, bloom, and shadow effects
